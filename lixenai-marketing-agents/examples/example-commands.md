# Example Commands

Use these with Codex or with individual agent files.

## Marketing Director Agent

```text
Use the Marketing Director Agent to create this week's marketing plan for recruiting founding LixenAI Agency Partners. Paid ads are not ready. Assets available: live demo, partner application, med spa niche snapshot, founder profile.
```

## Campaign Agent

```text
Create a 14-day LinkedIn campaign to recruit founding LixenAI Agency Partners.
```

## Partner Acquisition Agent

```text
Create 10 LinkedIn DM scripts for high-ticket closers. Filter for sales ability, consistent weekly effort, and comfort selling the med spa production niche.
```

## Copywriting Agent

```text
Write a landing page section explaining what the partner owns vs what LixenAI owns.
```

## Ads Manager Agent

```text
Write 5 ad angles for sales professionals who want to sell AI automation without building the tech. Proof is limited, so include a go/no-go recommendation.
```

## Social Media Agent

```text
Create a weekly content calendar for LixenAI partner recruitment across LinkedIn, Instagram, TikTok, and YouTube Shorts.
```

## Sales Enablement Agent

```text
Create a 90-second demo script for missed-call recovery.
```

## QA Review

```text
Review this ad copy for risky claims and rewrite it so it follows the LixenAI compliance rules.
```
