# Campaign Production Workflow

## Purpose

Use this workflow to turn one LixenAI business goal into a campaign package with copy, social content, ad drafts, outreach, and sales enablement.

## Multi-Agent Flow

### Step 1: Marketing Director Receives Business Goal

Input:

```text
Business goal:
Audience:
Offer:
Current launch stage:
Assets available:
Readiness status:
Timeline:
```

Marketing Director decides whether the campaign is partner-facing, client-facing, or internal.

### Step 2: Campaign Agent Creates Campaign Brief

Campaign Agent defines:

- Campaign name
- Objective
- Target audience
- Core problem
- Core transformation
- Offer angle
- Message pillars
- Channel plan
- Required assets
- CTA
- Measurement plan
- Compliance notes

### Step 3: Copywriting Agent Writes Core Copy

Copywriting Agent produces:

- Headline set
- Landing page section
- CTA set
- Short sales letter
- Founder post
- Objection-handling copy

### Step 4: Social Media Agent Repurposes The Campaign

Social Media Agent creates:

- 5 LinkedIn posts
- 1 carousel outline
- 3 short-form video scripts
- Caption pack
- Hook list

### Step 5: Ads Manager Agent Reviews Paid Readiness

Ads Manager Agent must answer:

- Is proof available?
- Is the funnel trackable?
- Is fulfillment ready?
- Are disclaimers needed?
- Should this run as paid, retargeting-only, or organic-first?

If readiness is not approved, Ads Manager creates ad angles for organic testing only.

### Step 6: Partner Acquisition Agent Builds Outreach Assets

For partner-facing campaigns, Partner Acquisition Agent creates:

- LinkedIn DM scripts
- Recruitment posts
- Application questions
- Follow-up messages
- Referral scripts

### Step 7: Sales Enablement Agent Builds Close Support

Sales Enablement Agent creates:

- Discovery script
- Demo script
- Objection library
- Close checklist
- Client handoff checklist

### Step 8: Marketing Director Final Review

Marketing Director checks:

- Strategic alignment
- Compliance
- Message consistency
- Funnel readiness
- Human review requirement

## Final Campaign Package

```text
Campaign brief:

Core copy:

Organic social assets:

Paid/retargeting recommendation:

Partner acquisition assets:

Sales enablement assets:

QA result:

Final recommendation:
```
