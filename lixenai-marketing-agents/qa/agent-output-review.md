# Agent Output Review

Use this review whenever one agent hands work to another agent or to the Marketing Director Agent.

## Review Questions

1. Does the output answer the requested task?
2. Is the audience named clearly?
3. Is the offer clear?
4. Is the CTA specific?
5. Are all claims supportable?
6. Does it use approved LixenAI language?
7. Does it avoid risky or unapproved language?
8. Does it correctly separate what partners own from what LixenAI owns?
9. Does it keep paid acquisition gated when readiness is missing?
10. Does it include human review before publishing?

## Review Output

```text
Status:
[approved, approved with edits, revise, reject]

Audience:
[partner-facing, client-facing, internal]

Strongest part:
[insert note]

Risks:
[insert risks]

Required edits:
[insert edits]

Next agent:
[insert agent or none]
```
