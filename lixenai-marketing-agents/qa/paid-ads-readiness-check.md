# Paid Ads Readiness Check

Use this before the Ads Manager Agent approves any paid campaign.

## Default Rule

Paid acquisition is not approved by default. If proof, funnel performance, case study, or fulfillment readiness is missing, recommend organic testing first.

## Readiness Questions

| Question | Pass/Fail | Notes |
| --- | --- | --- |
| Is the funnel live and trackable? |  |  |
| Is the landing page approved? |  |  |
| Is the partner or client offer clear? |  |  |
| Is fulfillment ready for the promise? |  |  |
| Is med spa the production niche for this campaign? |  |  |
| Are proof points real and supportable? |  |  |
| Is there a case study or validated result? |  |  |
| Are disallowed claims removed? |  |  |
| Is the income disclaimer included when needed? |  |  |
| Has a human reviewed the ad? |  |  |

## Recommendation Options

- Approved for limited paid test
- Retargeting-only draft
- Organic-first test
- Not approved

## Output Format

```text
Paid ads recommendation:
[approved for limited paid test, retargeting-only draft, organic-first test, not approved]

Reason:
[insert reason]

Required before launch:
[insert requirements]

Safer next step:
[insert organic or retargeting alternative]
```
