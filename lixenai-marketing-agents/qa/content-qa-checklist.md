# Content QA Checklist

Run this checklist before any LixenAI asset is published, sent, launched, or added to a partner playbook.

## Required Checks

| Check | Pass/Fail | Notes |
| --- | --- | --- |
| Clear audience |  | Is this partner-facing, client-facing, or internal? |
| Clear CTA |  | Is the next action direct and trackable? |
| Correct LixenAI positioning |  | Does it align with partner-first growth and done-for-you deployment? |
| Approved language used |  | Agency Partner, AI Growth System, live demo, niche snapshot, fulfillment backend, etc. |
| No unsupported income claims |  | No unverified financial outcomes. |
| No guaranteed earnings |  | No guaranteed income, revenue, or partner success. |
| No guaranteed appointments |  | No guarantee of bookings or clients. |
| No employment implication |  | No salary, wage, job, or employment framing for Agency Partners. |
| Partner opportunity language included |  | Use "independent, performance-based partner opportunity" when describing the program. |
| No unsupported healthcare compliance claims |  | No "fully HIPAA compliant" unless legal/compliance review approves it. |
| Med spa positioned correctly |  | Med spa is the current production niche. |
| Future niches positioned correctly |  | Other industries are future or conditional unless approved. |
| Paid acquisition readiness checked |  | Paid ads are not approved before proof, funnel data, and fulfillment readiness. |
| Human review required |  | A human must review before publishing. |

## Automatic Fail Conditions

Fail the asset if it contains any of the following:

- Passive income framing
- Guaranteed earnings
- Guaranteed appointments
- Guaranteed first client
- Guaranteed revenue
- No work required
- Done-for-you clients
- Employment, salary, or wage framing for Agency Partners
- Fully HIPAA compliant without approval
- Every niche is ready
- Paid ads are ready without proof, funnel data, and fulfillment readiness

## QA Output Format

```text
QA result:
[pass, needs revision, fail]

Audience classification:
[partner-facing, client-facing, internal]

Primary CTA:
[insert CTA]

Risk notes:
[insert notes]

Required revisions:
[insert revisions]

Human review:
Required before publishing.
```
